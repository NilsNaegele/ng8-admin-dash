Wie nutzen

Um einen Knopf mit farbigen Hintergrund zu verwenden, nutze den CSS Klassen Namen mit der korrespondierenden Farbe, kleingeschrieben mit dem Präfix button--colored-.
Wenn der Farb Name aus mehreren Wörtern besteht, sind die Worte mit einem minus "-" getrennt.
Hier kannst Du die Farbauswahl einsehen hier.

Beispiel: Du möchtest den farbigen Knopf mit dem Farbnamen "Rot" nutzen. CSS Klasse für diesen Fall ist .button--colored-red


Um die sozialen Knöpfe zu nutzen, benutze die CSS Klasse zum korrespondierenden Sozialen Netzwerk kleingeschrieben mit dem Präfix button--.

Beispiel: Du möchtest den Facebook Knopf nutzen. CSS Klasse für diesen Fall ist .button--facebook
